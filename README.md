# انا اسف علي كل حاجه زعلتك او ديقتك عمري ما ضحكت عليكي ف حاجه قلتها ليكي والله
# كان نفسي اقلك ان ساعات بتكون خايف توعد حد بحاجه وحد يجي ياخدها منك لانك مش جاهز ليسا وساعتها تبقا انتي يا توافق بالواقع يا ترفض ويكون الرفض دا يوقعك في مشكله

عرفني ان كان في حد متسبنيش ماسك في قشيه 
دا انت وعدتني وقلتي
مش هسيبك طول عمري 
مش لاقي دلوقتي
